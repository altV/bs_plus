# BsPlus

CLI for browserstack to use automate plan from command line.

## Installation

Add this line to your application's Gemfile:

    gem 'bs_plus'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install bs_plus

## Usage

Currently there's a binary named bs, it prints its usage.
